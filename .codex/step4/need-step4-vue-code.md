# 校园设施报修与资产管理系统第四阶段 Vue3 前端设计

> 本文档只明确第四阶段前端核心页面与交互。

## 1. 前端规范

- Vue 文件只保留 `<template>` 与外部资源声明。
- 脚本使用同名 `.js`，有私有样式时使用同名 `.scss`。
- 页面统一通过 `src/api` 调用后端。
- 状态文本、标签和筛选选项使用共享枚举。
- 前端控制入口和交互，后端负责最终权限、状态和并发校验。
- 异步页面处理加载、空数据、错误和无权限状态；提交期间禁用重复操作。

## 2. 资产页面与批量导入

扩展资产列表与详情：

- 管理员查看和编辑购入日期。
- 管理员查看动态计算的已购入年数和月数。

新增管理员资产导入页面：

```text
views/admin/asset-import
views/admin/asset-import/detail
```

- 批次列表展示文件名、上传人、上传时间和各卡片状态数量。
- 上传仅接受 `.xlsx`，提示单次最多 500 行，上传期间禁用重复提交。
- 批次详情展示待审核、已确认和已忽略卡片，并支持按状态筛选。
- 卡片展示资产名称、分类文本、购入日期、位置文本、疑似重复提示和失败原因。
- 待审核卡片支持编辑、忽略、单项确认和批量确认。
- 编辑卡片时补充正式资产所需的分类和结构化位置信息。
- 已确认和已忽略卡片只读，不显示重复操作入口。
- 批量确认后分别展示成功与失败结果。
- 删除未完成批次前二次确认。
- 后端返回状态冲突时刷新卡片数据并展示明确原因。

## 3. 维修师傅接单状态

维修师傅个人页面新增：

- 展示当前接单状态。
- 支持本人暂停或恢复接单。
- 暂停时填写必填原因和可选预计恢复时间。
- 恢复接单前二次确认，成功后刷新抢单大厅数据。
- 暂停期间仍正常展示和处理已有工单。
- 暂停状态下隐藏或禁用抢单大厅入口。

管理员维修师傅列表：

- 只读展示接单状态、暂停原因和预计恢复时间。
- 不提供修改接单状态入口。
- 支持按接单状态筛选维修师傅。

## 4. 账号人工恢复

新增管理员页面：

```text
views/admin/manual-account-recovery
```

- 列表支持按目标用户、状态和创建时间查询。
- 支持创建普通用户恢复申请、查看详情和审批。
- 创建申请时选择目标用户，并填写新手机号和线下核验说明。
- 发起管理员不能审批本人创建的申请。
- 仅待复核申请显示审批入口，发起管理员可撤销本人未审批申请。
- 审批处理说明必填。
- 展示三天验证期限和过期状态。
- 不展示密码、完整验证码或其他认证秘密。

新增用户新手机号验证页面：

- 仅允许处理已通过且未过期申请。
- 支持发送并校验绑定人工恢复场景的新手机号验证码。
- 完成后提示重新登录。
- 过期、已完成或状态变化时展示后端返回原因，不允许继续提交。

## 5. 管理统计与导出

扩展管理员统计页面：

- 分区展示维修效率、资产维修和维修师傅客观工作统计。
- 维修效率展示已完成数量、平均首次接单时长、平均处理时长、平均完成时长及超时数量。
- 资产维修统计展示维修次数较多资产、资产分类报修数量及购入时间较长且维修较多资产。
- 维修师傅统计展示接单状态、接单数、完成数和当前处理中数量。
- 支持最近 7、30、90 天和本年度，默认 30 天。
- 支持进入资产详情查看维修历史。
- 支持导出当前统计结果，导出期间禁用重复操作。
- 不展示排名、评分、风险等级或自动处置入口。

维修师傅个人统计页面：

- 仅展示本人接单状态、接单数、完成数、处理中数量和平均时长。
- 不展示排名、评分或其他维修师傅数据。

## 6. API 模块

建议新增：

```text
api/asset/import.js
api/admin/manual-account-recovery.js
api/statistics/management.js
```

扩展：

```text
api/asset/index.js
api/repairer/index.js
```

## 7. 路由与权限

- 资产导入、人工恢复和全局管理统计路由仅管理员可见。
- 维修师傅接单状态与个人统计路由仅维修师傅可见。
- 学生和教师不显示第四阶段管理功能入口。
- 页面按钮权限与状态仅改善交互，所有请求仍以后端鉴权结果为准。

## 8. 前端验证

- 生产构建通过。
- 验证资产导入上传限制、卡片审核、部分成功、重复操作和错误提示。
- 验证维修师傅本人修改接单状态、暂停后抢单限制及管理员只读权限。
- 验证人工恢复创建、双管理员审批、手机号验证和过期展示。
- 验证统计角色权限、时间范围和导出交互。
